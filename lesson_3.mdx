<h2 style="text-align: center;"> Αντιστάσεις </h2>

Οι αντιστάσεις (Resistors) χρησιμοποιούνται για να μειώσουν την ποσότητα ρεύματος η οποία πηγαίνει σε ένα εξάρτημα του κυκλώματός μας. 
Ποιο συχνές χρήσεις είναι σε αισθητήρες υγρασίας και LEDs. 
Υπάρχουν πολλά είδη αντιστάσεων όχι ως προς την κατασκευή αλλά ως προς το ποσό ρεύματος που αφήνει να περάσει.

[Τιμές αντιστατών](https://power.mozello.com/posts/params/post/1511923/15)

<h2 style="text-align: center;"> Δίοδοι </h2>

Αν και δε θα χρησιμοποιήσουμε τις διόδους στο μάθημα,
είναι κάτι σχετικό με τις αντιστάσεις, 
οπότε και θα κάνουμε αναφορά στις διόδους εδώ.

Οι δίοδοι επιτρέπουν στο ηλεκτρικό ρεύμα να περάσει από τη μια κατεύθυνση, 
και παράλληλα μπλοκάρουν την κίνηση από την αντίθετη κατεύθυνση. 
Οπότε, τα κυκλώματα που απαιτούν ροή προς μία μόνο κατεύθυνση περιλαμβάνουν μία ή περισσότερες διόδους στη σχεδίαση του κυκλώματος.

<h2 style="text-align: center;"> LEDs </h2>

Τα LEDs (Light-Emitting Diode), είναι ένας ημιαγωγός ο οποίος εκπέμπει φωτεινή ακτινοβολία στενού φάσματος. 
Συνήθως, τα LEDs χρειάζονται αντιστάσεις 200Ω - 220Ω, διότι σε διαφορετική περίπτωση θα καούν στιγμιαία.
Έχουν δύο άκρα (ποδαράκια), ένα για το GND (γείωση)(το κοντό) και ένα για τη σύνδεση με ένα digital pin (το μακρύ).

---

<h2 style="text-align: center;"> Σύνδεση ενός LED στο breadboard </h2>

Για να συνδέσουμε το LED με το arduino πρέπει:
- Να βάλουμε τα ποδαράκια του LED σε δύο οποιεσδήποτε θύρες στο breadboard.
- Στην ίδια κάθετη σειρά με το αρνητικό ποδαράκι (το κοντό) συνδέουμε ένα jumper wire που καταλήγει στο GND του Arduino.
- Στην ίδια κάθετη σειρά με το θετικό ποδαράκι (το μακρύ) συνδέουμε τη μία άκρη της αντίστασης των 220Ω.
- Στην ίδια κάθετη σειρά με την άλλη πλευρά τις αντίστασης συνδέουμε ένα καλώδιο που καταλήγει σε ένα από τα digital pins 2-13.

Ας δούμε τη σύνδεση σε schematics:

[Schematics](https://cloud-brw8sh1bt-hack-club-bot.vercel.app/0mighty_turing-leelo.png)

<h2 style="text-align: center;"> Κώδικας </h2>

Για να λειτουργήσει ένα λαμπάκι,
θα πρέπει να περάσουμε στο Arduino μας έναν απλό κώδικα.
Αυτός ο κώδικας θα κάνει το λαμπάκι μας να αναβοσβήνει.

Ας δούμε τον κώδικα:

```cpp
void setup() {
    pinMode(5, OUTPUT); // Όρισε το πιν 5 ως έξοδο
}

void loop() {
    digitalWrite(5, HIGH); // Άναψε το LED
    delay(1000); // Περίμενε 1 δευτερόλεπτο εδώ
    digitalWrite(5, LOW); // Σβήσε το LED
    delay(1000); // Περίμενε 1 δευτερόλεπτο εδώ
}
```
