<h2 style="text-align: center;"> Motors </h2>

Οι κινητήρες του Arduino βοηθάνε σε διάφορα project που χρειάζονται κίνηση. Συνδέονται με 3, 4, 5 ή 6 pins. 
Για μεγαλύτερη ευκολία σε κάποια project είναι προτιμότερο ή αναγκαίο να χρησιμοποιήσουμε motor shields.
Υπάρχουν 2 βασικά είδη από motors:
Step Motor - Εκτελεί μέγιστη περιστροφή 360 μοιρών.
Servo Motor - Εκτελεί μέγιστη περιστροφή 180 μοιρών.

<h2 style="text-align: center;"> Arduino Shields </h2>

Υπάρχουν μικρές πλακέτες κυκλωμάτων που μπορούν να χρησιμοποιηθούν με το Arduino για να βελτιώσουν τις λειτουργίες των πλακών Arduino. Αυτές οι πλακέτες ονομάζονται ασπίδες (shields), τοποθετούνται στις πλακέτες Arduino και είναι παρόμοιες με τις πλακέτες Arduino. 

Οι ασπίδες Arduino έχουν ακριβώς τις ίδιες ακίδες με αυτές στις πλακέτες Arduino και μόνο σε αυτή την περίπτωση οι ασπίδες μπορούν να τοποθετηθούν στις πλακέτες Arduino. Για την εκτέλεση διαφορετικών λειτουργιών όπως η λειτουργία ορισμένων κινητήρων ή η παροχή πρόσβασης στο Διαδίκτυο στο Arduino και πολλές άλλες. 

Η χρήση ασπίδων είναι συνηθισμένη επειδή:
1. Αποσπώνται εύκολα από τις πλακέτες Arduino.
2. Δεν απαιτούνται καλώδια σύνδεσης για τη σύνδεση των ασπίδων στο Arduino.
3. Οι ασπίδες μπορούν να χρησιμοποιηθούν με το Arduino τοποθετώντας τις το ένα πάνω στο άλλο.
4. Κάνουν επίσης το κύκλωμα να φαίνεται προσεγμένο και συμπαγές.

Υπάρχουν 5 διαφορετικοί τύποι ασπίδων που συζητούνται σε αυτήν την ομιλία που χρησιμοποιούνται στις πλακέτες Arduino:

* Ασπίδα κινητήρα για Arduino.
    Η θωράκιση κινητήρα χρησιμοποιείται με το Arduino όταν πρόκειται να ελεγχθούν κινητήρες συνεχούς ρεύματος ή step motors. 
    Συνοδεύεται από ψήκτρα που βελτιώνει την απόδοση της θωράκισης.

* Ασπίδα οθόνης Liquid Crystal display (lcd) 16×2.
    Αυτή η ασπίδα διαθέτει οθόνη LCD 16×2 με 6 κουμπιά που μπορούν να χρησιμοποιηθούν για εναλλαγή στο μενού ή για επιλογή μιας επιλογής από το μενού. 
    Αυτές οι ασπίδες χρησιμοποιούνται κυρίως σε έργα ή συσκευές όπου πρέπει να περάσουμε από διαφορετικές επιλογές σε διαφορετικές χρονικές στιγμές.

* Ασπίδα relay για το Arduino.
    Οι ασπίδες relay χρησιμοποιούνται με το Arduino όταν πρόκειται να συνδεθούν συσκευές υψηλής τάσης και ρεύματος στις πλακέτες Arduino.
    Επιπλέον, αυτή η ασπίδα απαιτεί εξωτερική παροχή ρεύματος.

* Ασπίδα πολλαπλών λειτουργιών για Arduino.
    Αυτές η ασπίδες μπορεί να χρησιμοποιηθούν σε έργα όπου πρόκειται να εκτελεστούν πολλαπλές λειτουργίες. 
    Για παράδειγμα, όπως η ανίχνευση της θερμοκρασίας και η εμφάνισή της σε οθόνη επτά τμημάτων και η χρήση της ασύρματης μονάδας για την αποστολή των δεδομένων σε οποιαδήποτε άλλη συσκευή.

* GSM ή GPRS Arduino ασπίδα.
    Αυτή η ασπίδα επιτρέπει στο Arduino να έχει πρόσβαση στο διαδίκτυο, να λαμβάνει και να στέλνει μηνύματα και κλήσεις. 
    Αυτή η ασπίδα συνοδεύεται από μόντεμ M10 τετραπλής ζώνης που υποστηρίζει τα πρωτόκολλα επικοινωνίας TCP, UDP και HTTP. 
    Υπάρχει μια υποδοχή SIM στην ασπίδα που χρησιμοποιείται για τη διασύνδεσή της με το δίκτυο κινητής τηλεφωνίας.


Αν και η χρίση των ασπίδων (όπως και βλέπετε) είναι πολύ χρήσιμη δε θα χρησιμοποιήσουμε κάτι τέτοιο ούτε στη λίστα εξαρτημάτων που χρειαζόμαστε αλλά και ούτε στο project μας για λόγους απλότητας του προγράμματος.
Φυσικά, σας συστήνουμε να ψάξετε και μόνοι σας παραπάνω πληροφορίες για τις ασπίδες.

<h2 style="text-align: center;"> Κώδικας </h2>

Με αυτό το πρόγραμμα ο κινητήρας κάνει μισή περιστροφή με ταχύτητα 15 RPM και περιμένει 1 δευτερόλεπτο πριν επανεκτελέσει το πρόγραμμα

```cpp
#include <Stepper.h>
#include <Servo.h>  // Συμπεριλαμβάνουμε την βιβλιοθήκη για τον κινητήρα

// ¨ορίζουμε τον αριθμό "βημάτων" ανά περιστροφή. ¨οσο μεγαλύτερο το νούμερο τόσο πιο ομαλά γίνετε η περιστροφή. Ένα ιδανικό νούμερο είναι το 2048.
const int stepsPerRevolution = 2048;
Servo myservo;  // Δημιουργούμε μεταβλητή για τον κινητήρα
// Αντιστοιχούμε τα pins του κινητήρα με τα pins του arduino.
Stepper myStepper(stepsPerRevolution, 8, 10, 9, 11);

void setup() {
  // Ορίζουμε την ταχύτητα του μοτέρ σε RPM (rotations per second = περιστροφές ανά δευτερόλεπτο)
  myStepper.setSpeed(15);
  myservo.attach(3);  // Συνδέουμε τον κινητήρα με το pin 3 ή 9
}

void loop() {
  // Κούνησε το μοτεράκι ένα βήμα προς την φορά των δείκτων του ρολογιού
  myStepper.step(stepsPerRevolution/0.5);
  delay(1000);
}
```
